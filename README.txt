-Written in Python 3.5.2.

-I used nltk to remove stop words and stem my documents. The following link contains instructions on how to downlaod and install: http://www.nltk.org/install.html

*************************************************************************************************
To Run the part one program, type in something following the format below (such as using MobaXterm):
*************************************************************************************************

python KMeansTweets.py "25" "InitialSeeds.txt" "Tweets.json" "tweets_k_means_output.txt"


*************************************************************************************************
Where "25" is the number of clusters, "InitialSeeds.txt" is the path to the seed data file used for the centroids, and "Tweets.json" is the data file path. 
The "tweets-k-means-output.txt" is the name of the output file you want to save your data as.
